# Sushi Restaurant Management System

## Giới thiệu
Dự án này mô phỏng hệ thống quản lý quán sushi:
- Quản lý thực đơn
- Quản lý bàn
- Đặt món & thanh toán

## Công nghệ
- Python 3.x
- Flask (REST API backend)
- SQLite (database)
- HTML/CSS (frontend đơn giản)

## Cài đặt
```bash
git clone https://github.com/<username>/sushi-restaurant-management.git
cd sushi-restaurant-management
pip install -r requirements.txt
python src/app.py
```
