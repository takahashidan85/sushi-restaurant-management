# 1. Giới thiệu
1.1 Mục tiêu: Xây dựng hệ thống quản lý quán sushi để quản lý thực đơn, bàn, và đặt món.
1.2 Phạm vi: Quản lý thông tin món ăn, bàn, đơn hàng, và thanh toán.

# 2. Yêu cầu chức năng
- Quản lý thực đơn (thêm/sửa/xóa món)
- Quản lý bàn ăn
- Tạo đơn hàng
- Tính hóa đơn

# 3. Yêu cầu phi chức năng
- Thời gian phản hồi dưới 1 giây cho truy vấn đơn giản
- Dễ mở rộng khi thêm tính năng mới
